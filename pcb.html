Custom boards for a custom operation. Designed in-house, fabricated through PCBway.

// BOARD 01 — BOTTOM BOARD

Main Controller

The bottom board carries both processors and all signal conditioning. MEGA2560 and ESP32 on the same board, communicating via serial at 19200 baud using the @XXX[data]! format. Level shifting between the 5V MEGA and 3.3V ESP32 is handled by TXS0108E bidirectional level shifters. Schmitt trigger buffers on all HMI inputs.

// BOARD 02 — TOP BOARD (HMI)

Human Interface Board

16×2 LCD, rotary encoder with pushbutton, two tact switches, piezo buzzer, three status LEDs. All inputs pass through Schmitt trigger buffers before reaching the ESP32 — no software debouncing needed.

// BOARD 01 — BOTTOM BOARD

Main Controller Board

The bottom board carries both processors and all signal conditioning. MEGA2560 and ESP32 share the board, communicating via serial at 19200 baud using the @XXX[data]! message format. Level shifting between the 5V MEGA and 3.3V ESP32 is handled by TXS0108E bidirectional level shifters. Schmitt trigger buffers on all HMI inputs for clean, bounce-free signals.

// BOARD 02 — TOP BOARD (HMI)

Human Interface Board

The top board carries the operator interface — 16×2 LCD, rotary encoder with pushbutton, two tact switches, piezo buzzer, and three status LEDs. All inputs pass through Schmitt trigger buffers before reaching the ESP32, eliminating switch bounce in hardware rather than software.

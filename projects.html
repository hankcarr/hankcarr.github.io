SMD Component Tape Cartridges

A parametric system for storing and dispensing surface mount component tape. Each cartridge holds a 6-foot length of tape coiled inside a resin-printed enclosure, with a tape slot sized precisely to the tape thickness — tight enough that components stay in place once the cover tape is peeled back, loose enough to feed smoothly.

Covers and cartridges connect using standard LEGO Technic pins, meaning no custom fasteners are needed. Cartridges connect to each other in any configuration — side by side, stacked, in a row — using the same pins and the Technic-compatible holes moulded into every face. Pull the whole gang out as a unit and every component for a project is right at hand.

Printed in ABS-like resin on an Elegoo Mars 3+ — the resin process delivers the dimensional accuracy needed for the tight tape slot tolerances, and the surface finish is far superior to FDM. The build plate texture gives each cartridge a subtle matte finish that looks intentional.

Colour Coding

Model Number System

Cartridge dimensions are encoded directly into the model number embossed on every unit: DD.WWTT where DD is roll diameter in mm, WW is tape width in mm, and TT is tape thickness in tenths of mm.

Model 60.1015
60mm roll · 10mm wide · 1.5mm thick · Standard 0805 tape
Model 60.1419
60mm roll · 14mm wide · 1.9mm thick · SOT-23 / wider components
Capacity
~500 components per cartridge at standard 4mm pitch
Tape length
6 feet per cartridge
Printer
Elegoo Mars 3+ MSLA resin
Material
ABS-like resin — tough and dimensionally stable
Fasteners
Standard LEGO Technic pins — no custom hardware
Storage
Plano StowAway Model 3743
